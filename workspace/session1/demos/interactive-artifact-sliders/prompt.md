# Demo: Interactive Artifact with Sliders and Dropdown

## What This Demonstrates
- Artifacts are not just charts — they are interactive web applications
- AI builds tools you USE, not just documents you READ
- Sliders, dropdowns, real-time calculations — zero coding
- The "wow" moment for the audience

## Tool
Claude.ai (new conversation)

## Prompt

> Build an interactive loan payment calculator as an artifact. Include:
> - A slider for loan amount ($50,000 to $1,000,000, step $10,000)
> - A slider for annual interest rate (3% to 12%, step 0.25%)
> - A dropdown menu to select loan term: 10, 15, 20, or 30 years
> - Display: monthly payment, total interest paid, and total cost
> - Include a donut chart showing principal vs. total interest that updates as you move the sliders
> - Clean, modern styling with a dark header

## Key Narration Points
- "This is a working web application — not a static image."
- "Drag the loan amount slider. Watch everything recalculate instantly."
- "Change the dropdown from 30 years to 15. See how the interest ratio changes."
- "One prompt. No coding. An interactive tool you could send to a client."
- "This is what Claude calls an 'artifact' — something you interact with, not just read."

## Why This Demo Matters
It shifts the audience's mental model from "AI generates text" to "AI builds tools." The loan calculator is universally understood — everyone has a mortgage or car loan. They immediately see how this applies: ROI calculators, pricing tools, scenario planners.

## Timing
~4 minutes (including audience interaction with sliders)
