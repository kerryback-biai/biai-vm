# Session 1 Demos: The AI Landscape

Instructor demos using Claude.ai (browser). These show Tier 1 capabilities.

## Demo 1: Chart from Pasted Data
Paste quarterly revenue data into Claude.ai. Ask for a chart. Show the artifact rendering and download.

## Demo 2: Interactive Artifact with Sliders
Ask Claude to build an interactive calculator with sliders as an artifact.

## Demo 3: Multi-Deliverable from Web Search
Ask Claude to research a topic, produce a summary, chart, and PowerPoint — all from one conversation.

## Demo 4: PowerPoint File Generation
Ask Claude to create a PowerPoint deck summarizing data — 3 slides, executive format. Download and open the .pptx file.
