# Demo: Multi-Deliverable with Web Search

## What This Demonstrates
- Web search for live/current data (not from training)
- Multiple output types from a single prompt (chart + document)
- Three capabilities in one shot: search, visualize, document

## Tool
Claude.ai (new conversation)

## Prompt

> Search the web for the current market capitalizations of Apple, Microsoft, Nvidia, Amazon, and Alphabet (Google). Then:
> 1. Create an interactive bar chart artifact comparing them, sorted largest to smallest
> 2. Create a one-page Word document (.docx) ranking them with a 2-sentence commentary on each company's position

## Key Narration Points
- "Three capabilities in one prompt: web search, interactive chart, downloadable document."
- "The data is current — it searched the web just now. Not from training data."
- "That Word doc is ready to email to your team. Real .docx file."
- "One prompt. Twenty dollars a month. Three deliverables."
