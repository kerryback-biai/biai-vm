# Demo: Chart from Pasted Data

## What This Demonstrates
- Tier 1 browser chatbot (Claude.ai) running Python automatically
- Artifact rendering (interactive chart in browser)
- Zero setup — paste data, get a chart

## Tool
Claude.ai (browser, live on projector)

## Data to Paste

```
Quarter, Revenue ($M), YoY Growth
Q1 2024, 3.2, --
Q2 2024, 3.5, --
Q3 2024, 3.8, --
Q4 2024, 4.1, --
Q1 2025, 3.9, 21.9%
Q2 2025, 4.3, 22.9%
Q3 2025, 4.7, 23.7%
Q4 2025, 5.1, 24.4%
```

## Prompt

> Here is quarterly revenue data. Create a bar chart showing revenue by quarter with a line overlay showing YoY growth rate on a secondary axis. Use a professional color scheme — dark blue bars, amber line.

## Key Narration Points
- "Notice it ran Python automatically — you didn't ask for code."
- "The chart is an artifact — interactive, hover for values."
- "You can download this as PNG or copy it into a deck."
- "This took about 15 seconds. How long in Excel?"
