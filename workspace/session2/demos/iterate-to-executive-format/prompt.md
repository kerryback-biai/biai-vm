# Demo: Iterate to Executive Format

## What This Demonstrates
- Specific direction produces specific improvement ("make it better" doesn't work)
- Reformatting for different audiences (analyst → manager → executive)
- Real file generation (Word document)

## Tool
Claude Code on AI+Code Lab (same session)

## Prompts (sequential iterations)

### Iteration 1: Visual Redesign
```
Change the monthly revenue trend to a stacked area chart showing all three product lines. Save as PNG.
```

### Iteration 2: Traffic-Light Format
```
Reformat the key metrics as a traffic-light dashboard — green if the metric is above the median across regions, yellow if within 10% of median, red if below. Display as a formatted table.
```

### Iteration 3: Executive Summary Document
```
Take the best charts and the key findings — including the concentration risk — and produce a one-page executive summary as a Word document (.docx). Lead with the biggest risk, end with three recommended actions.
```

## Key Narration Points
- "Specific direction: 'stacked area chart showing all three product lines.' Not 'make it better.'"
- "Traffic-light is what executives actually want — not raw numbers."
- "That Word doc is a real .docx — download it, open it, send it to your board."
- "Same data, three iterations, dramatically different output each time."

## Timing
~3 minutes (do iteration 3 live; show 1-2 as pre-done examples if time is short)
