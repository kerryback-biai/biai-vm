# Session 2 Demos: AI vs Dashboards

All demos run on AI+Code Lab (Claude Code) using data in `data/`.

## Demo 1: Dashboard Replication
Replicate the traditional Acme Corp dashboard — revenue by product line, region, top customers, monthly trend, headcount, key metrics. Same output, minutes instead of days.

## Demo 2: Ad-Hoc Beyond Dashboard
Ask questions the dashboard was never designed for — customer decline trends, concentration risk, revenue per rep, churn signals, H1 2024 vs H1 2025 comparisons.

## Demo 3: Iterate to Executive Format
Transform raw output into executive-ready deliverables — stacked area charts, traffic-light dashboards, Word documents.

## Demo 4: Self-Critique
Ask Claude to critique its own work: "What would a skeptical CFO question?" Reveals assumptions about churned customers, ambiguous metrics, and data pipeline issues.
