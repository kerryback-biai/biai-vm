# Session 6 Demos: Decision-Making with AI

All demos run on AI+Code Lab (Claude Code) using portfolio data in `data/portfolio/`.

## Demo 1: Portfolio Optimization (Constrained)
Describe objectives and constraints in English. Claude writes mixed-integer optimization code (PuLP/scipy). Shows that changing priority order changes the answer.

## Demo 2: What-If Scenarios
Four scenarios in seconds: assess current state, sell all Sell-rated, sell only cheap lots, plot the tax-cost vs. sector-alignment tradeoff curve.

## Demo 3: AI as Discussion Partner (Multi-Objective Tradeoffs)
Conversational exploration: "Walk me through why you'd hold AAPL despite a Sell rating." "What if the client needs college money in 5 years?" "Show me conservative vs. aggressive plans side by side."
