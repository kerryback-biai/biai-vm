# Session 7 Demos: Predictive Modeling

## Demo 1: Churn Classification
Build a gradient boosting model from English instructions. Shows confusion matrix, feature importance, cross-validation. Data: `data/customer_churn.csv` (7,043 telecom customers).

## Demo 2: Model Evaluation
Precision-recall tradeoff at different thresholds (0.3/0.5/0.7). Business cost analysis: retention offer cost vs. revenue saved. Where the model fails — blind spots and missing data.

## Demo 3: Regression — Demand Forecast
Same gradient boosting method applied to regression (predicting monthly revenue from Session 2's Acme Corp sales data). Shows that with only 18 months of data, simpler methods may be better.
